# 小程序生命周期钩子执行顺序


```
onReady: function () {
    // Do something when page ready.
    console.log('onReady')
  },
  onShow: function () {
    // Do something when page show.
    console.log('onShow')
  },
  onHide: function () {
    // Do something when page hide.
    console.log('onHide')
  },
  onUnload: function () {
    // Do something when page close.
    console.log('onUnload')
  },
  onLoad: function () {
    console.log('onLoad')
  }
```

测试发现顺序:
```
onLoad
  ↓
onShow
  ↓
onReady
```
